Responsive Design and Fluid Layouts

Description about the site here

By: Michael Teevan and Nate Beers

Responsive web design is making your web site respond to different browsers, devices, and sizes. It can resize depending on the zoom, or if the user resizes their browser, the page should automatically resize to fit the new size. With increasing amounts of large-resolutions screens on the market, web designers must take this into consideration when creating a design that is responsive to this layout.

In a fluid layout, web designers will typically give sections percentage heights/widths instead of exact pixel measurements. This makes the web site more user-friendly and more appealing to the eye, no matter the screen resolution.

Overall, I think it is very important to use fluid and responsive design. Today, there are so many devices and screen resolutions that people use to access the internet and we, as web designer, must try to accommodate as many of these people as possible.

  • How was it to implement these concepts?

    • Implementing these concepts went fairly straightforward. Using percentages instead of pixels for width and height felt natural. The different objects on the page seemed to fall together much easier than when I was using pixels.

  • How do you think smartphones and tablets have changed web design? How do you think new technology will change web design in the future?

    • Smartphones and tablets have changed web design by offering a multitude of different size screens with their own screen resolutions. Instead of creating a different website for each screen, developers adapted to using responsive web design and media queries in order for their web application to quickly adapt between devices.

  • How do you think visually impaired people use the web? Can they? Should we design and development for them?

    • There is a wide range of what is considered visually impaired, so instead of developing and designing for each group, I think the best way to handle it is allowing your web app to easily manipulated by other programs or browsers that are designed and developed for the specific purpose of assisting those groups who are visually impaired.
